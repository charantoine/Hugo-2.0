## Origine du code, provenance des sources et clarification de contribution

Le présent dépôt contient un travail de transformation, de convergence, de documentation et de développement local réalisé à partir d’un socle technique initialement développé par **Pierre Fardeau** (compte GitHub : [hamanad](https://github.com/hamanad)).

### 1. Provenance du socle initial

Le code de départ utilisé pour ce travail ne résulte pas d’un développement ex nihilo dans ce dépôt.  
Il provient de briques téléchargées en **archives ZIP entre le 11 juin 2026 et le 12 juin 2026** depuis le dépôt privé de Pierre Fardeau.

Ces archives ZIP ont servi de base de travail technique pour les développements réalisés ensuite en local.  
Afin de rendre cette provenance explicite et vérifiable, les ZIP de départ ont été **conservés à la racine du présent dépôt**.

### 2. Modalité de travail

À partir de ce socle initial téléchargé en ZIP, le travail a été mené **localement sur Mac**, sous forme :
- d’analyses et d’audits du code existant ;
- de documentation de convergence ;
- d’ajouts, corrections et transformations progressives ;
- de tests et de structuration documentaire associés.

L’état consolidé de ce travail local a ensuite été **déposé sur GitHub le 19 juin 2026** dans le présent dépôt, afin de permettre une revue claire du contenu, de ses écarts, de ses propositions de réutilisation et des éléments à reprendre ou à refaire.

### 3. Clarification de propriété et d’attribution

Afin de lever toute ambiguïté sur la propriété intellectuelle, la paternité technique et la nature des contributions, il convient d’entendre explicitement que :

- le **socle de code initial** utilisé comme point de départ relève du travail de **Pierre Fardeau** ;
- le **présent dépôt** contient ensuite un travail dérivé de transformation, de convergence, de développement local, d’audit, de test et de documentation réalisé à partir de cette base ;
- la présence de ce code dans le présent dépôt **ne doit pas être interprétée comme une réattribution de la paternité du socle initial** ;
- la conservation des archives ZIP à la racine du dépôt et la référence explicite au compte GitHub **hamanad** visent précisément à maintenir cette traçabilité et cette clarté d’attribution.

### 4. Finalité du dépôt

Le présent dépôt a pour objet de fournir une base de travail lisible et partageable permettant :
- d’identifier ce qui relève du socle initial ;
- d’identifier ce qui a été modifié, ajouté ou restructuré localement ;
- d’évaluer ce qui peut être réutilisé, repris, réintégré ou refait dans d’autres contextes de développement.

Ce dépôt doit donc être lu comme un **support de convergence et de revue technique**, construit à partir d’un socle clairement attribué à **Pierre Fardeau**, puis prolongé par un travail local documenté et partagé au **19 juin 2026**.
